# weather-app

![Screenshot](screenshot.png)

Aplicação web que mostra o clima atual usando geolocalização automática ou busca manual de cidades.

**Demo:** [https://weather-app-eight-ashen-97.vercel.app/](https://weather-app-eight-ashen-97.vercel.app/)

## Resumo

Criei esta aplicação para praticar consumo de APIs REST com JavaScript puro. Utilizo a WeatherAPI.com para mostrar:
- Localização atual (geolocalização HTML5)
- Temperatura em °C
- Sensação térmica
- Cidade e região
- Busca por qualquer cidade mundial

Aprendi a ler documentação de API, tratar erros CORS e fazer deploy em produção.

## Funcionalidades
- Geolocalização automática (detecta Natal RN)
- Busca cidades em português e inglês
- Design responsivo mobile-first
- HTTPS production-ready

## Ferramentas
- HTML5
- CSS3 (glassmorphism)
- Vanilla JavaScript
- WeatherAPI.com REST API
- GitHub Pages + Vercel
